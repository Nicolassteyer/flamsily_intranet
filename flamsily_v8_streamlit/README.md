# Flamsily V8

Application Streamlit premium pour la communauté Flam's.

## Lancer en local
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Déployer sur Streamlit Cloud
- Dépôt GitHub avec les fichiers à la racine
- Main file path : `app.py`

## Compte admin de démo
- Email : `admin@flamsily.local`
- Mot de passe : `admin12345`

## Fonctionnalités
- Connexion / inscription sur une seule URL
- Arrivée directe sur le fil d'actualités
- Accueil résumé
- Annuaire avec recherche sur tous les champs
- Événements
- Carrières
- Messages visible en placeholder
- Profil modifiable
- Paramètres
- Admin : actualités, événements, carrières, membres

## Notes
- Le reset e-mail est prévu pour une version ultérieure
- La base SQLite est créée automatiquement dans `data/flamsily.db`
