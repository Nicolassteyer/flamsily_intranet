
# Flamsily V4

Web app Streamlit plus directe et plus fonctionnelle :

- page d'entrée = connexion / inscription
- une seule URL
- espace membre
- actualités
- annuaire
- événements
- carrières
- administration

## Lancer en local

```bash
pip install -r requirements.txt
streamlit run app.py
```
