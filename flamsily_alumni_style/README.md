# Flamsily — style alumni

Version plus pro de l'intranet Flamsily.

## Fonctionnement
- une seule URL principale
- les utilisateurs s'inscrivent directement
- puis se connectent avec e-mail + mot de passe
- ensuite ils accèdent à l'intranet

## Lancer
```bash
pip install -r requirements.txt
streamlit run app.py
```
