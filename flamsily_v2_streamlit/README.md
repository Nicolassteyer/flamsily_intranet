# Flam'sily V2

Intranet Streamlit inspiré d'une interface type alumni, avec :

- URL unique
- inscription directe
- connexion e-mail + mot de passe
- accueil
- actualités
- événements
- annuaire
- carrières
- profil membre
- espace admin

## Lancer en local

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Compte admin de démonstration

- E-mail : `admin@flamsily.local`
- Mot de passe : `admin12345`

## Déploiement Streamlit Cloud

- Repository : ton repo GitHub
- Branch : `main`
- Main file path : `app.py`
