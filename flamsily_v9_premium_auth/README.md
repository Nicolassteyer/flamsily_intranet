# Flamsily V9

Application Streamlit prête à déployer pour l'intranet Flam's.

## Lancer en local

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Déploiement Streamlit Cloud

- Dépôt GitHub avec tous les fichiers à la racine
- Main file path : `app.py`
