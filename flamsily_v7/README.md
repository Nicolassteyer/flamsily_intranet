# Flamsily V7

Intranet premium Streamlit pour la communauté Flam's.

## Lancer en local

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Déploiement Streamlit Cloud

- Repository: votre repo GitHub
- Branch: `main`
- Main file path: `app.py`
